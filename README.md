# Nakib360's Personal Portfolio Website

Hey, I am Nakib360. Welcome to my Personal Portfolio Website — a modern, responsive, and interactive website that showcases my skills, projects, and achievements as a developer.

## Live Demo 
[View Portfolio](https://nakib-360.web.app/)

## About This Project
This portfolio is designed to represent my professional identity on the web development. It highlights my expertise, technical skills, and recent works in a clean and minimal design.

### Features
- Fully responsive for all devices  
- Modern UI built with HTML, CSS, and JavaScript, React, Tailwind, Framer-motion, ShadcnUI, Lucide. 
- Smooth animations and transitions  
- Dynamic project section  
- Dynamic skill section 
- Dynamic services section
- Contact form with email integration  

## Built With
- Frontend: HTML, CSS, JavaScript  
- Framework: React.js  
- Styling: Tailwind CSS
- Hosting: Firebase
